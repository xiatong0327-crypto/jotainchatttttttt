# Dual-Mac smoke checklist (v0.1.5+)

Use **two Apple Silicon Macs**, same Wi‑Fi, both on the **same version** build.

## Setup

- [ ] Install zip → Open-Me-First / right-click Open on both
- [ ] Local Network allowed for jotainchatttttttt (both)
- [ ] Both show in Devices; wait for **connected (green)**

## Chat

- [ ] Text both directions
- [ ] Sound (if enabled) on incoming text
- [ ] After sleep/wake: reconnect without manual reinstall

## Files (Accept required)

- [ ] File button → small file → **Accept** → completes under Downloads/jotainchatttttttt
- [ ] Drag file into chat → **Accept** → Open / Show in Finder work
- [ ] Image file via drag → still needs Accept (not auto)
- [ ] Reject / Cancel behave as expected

## Screenshot paste (auto)

- [ ] ⌘V screenshot ≤2 MB → receiver **no Accept** → preview in bubble
- [ ] Open / Show in Finder on screenshot card
- [ ] Screenshot &gt;2 MB refused or not auto (send via File instead)

## Resume (optional)

- [ ] Large file → interrupt mid-way → Resume or auto-resume
- [ ] Checksum OK on complete

## History / update

- [ ] Messages still present after quit/relaunch
- [ ] Replace .app with newer build → same history (`messages.db` path unchanged)

## Diagnostics

- [ ] Settings → Diagnostics shows codes when you force a failure (optional)
